Use this service to report a test result to the NHS after using a rapid lateral flow test kit for coronavirus (COVID-19).

Report the result:

  • every time you use a rapid lateral flow test kit (positive, negative, or void)
  • within 24 hours of getting the result

You can only report one result at a time.

Before you start

To use this service, you need:

  • the QR code or ID number printed on the test strip (the part of the kit that shows your result)
  • a mobile phone number so we can text you to confirm we've got your result
  • the NHS number for the person taking the test (called a CHI number in Scotland and H&C number in Northern Ireland). This is optional.
